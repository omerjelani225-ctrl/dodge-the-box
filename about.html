About

Dodge the Box: 100 Stages

Dodge the Box is a small HTML5 arcade game you can play in a browser. Your goal is simple: survive each stage by dodging falling blocks.

The game is designed to be lightweight, mobile friendly, and easy to publish on GitHub Pages.
